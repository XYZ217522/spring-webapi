### Spring-webapi

### before start
* git clone repo
* copy ```.env.sample``` to ```.env```
* start docker container ```docker-compose up -d```

### develop by docker
* enter a running container ```docker exec -it springboot-app sh```
* start springboot app ```./start_sever.sh```
